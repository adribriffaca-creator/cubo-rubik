# 🧊 Simulador Interactivo de Cubo Rubik 3x3

Un simulador de Cubo Rubik premium desarrollado en **Python** utilizando **PyQt6** y **OpenGL**. Diseñado específicamente para una experiencia fluida y estética en **Fedora Linux**.

![Captura del Simulador](assets/cubo-rubik.png)

## ✨ Características

* **Renderizado 3D Realista:** Motor gráfico basado en OpenGL con iluminación y sombras suaves.
* **Cronómetro Integrado:** Mide tu tiempo con precisión de centésimas.
* **Historial de Movimientos:** Registro en tiempo real de cada giro realizado.
* **Controles Personalizables:** Configura tus propios atajos de teclado.
* **Sistema de Mezclado:** Algoritmo de mezcla aleatoria de 20 movimientos.
* **Estética Premium:** Paleta de colores Dark Mode (Slate & Sky Blue).
